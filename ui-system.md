# UI System – Sorbit

## 1. Color Palette

- **Primary Color**: Blue (#007BFF)  
  Used for primary actions, buttons, and links.  
- **Accent Color**: Green (#4CAF50)  
  Used sparingly for highlights—key actions, toggles.  
- **Base Color**: Black (#111111)  
  Used for text, icons, and subtle dividers.  
- **Background**: White (#FFFFFF)  
  Used for backgrounds and containers to keep the interface clean.

## 2. Typography

- **Primary Font**: Open Sans (sans-serif)  
  Used for all body text, ensuring readability at small sizes.  
- **Secondary Font**: Lato (sans-serif)  
  Used for headings and key labels, adding contrast and clarity.

## 3. Spacing & Layout

- **Gutter (horizontal spacing)**: 16px between sections  
- **Vertical spacing**: 24px between major elements (e.g., buttons, inputs)  
- **Button padding**: 12px vertical, 24px horizontal  
- **Card margin**: 16px between cards

## 4. Components

- **Buttons**:  
  Rounded corners, filled blue for primary, green accent for success, subtle hover shadow effect.

- **Cards**:  
  Soft rounded edges, white background, slight shadow, blue heading.

- **Inputs**:  
  Minimal borders, subtle focus effect (light blue outline), large enough for easy tapping.

## 5. Buttons & Interactions

- Primary Action: Bold blue button, white text.  
- Secondary Action: White button, blue border, blue text.  
- Hover State: Subtle shadow grows, color shifts slightly lighter.

## 6. Accessibility Considerations

- Sufficient color contrast for all text against backgrounds.  
- Interactive elements (buttons, links) easy to tap, large hit areas.  
- Responsive design—flows seamlessly across desktop and mobile.
